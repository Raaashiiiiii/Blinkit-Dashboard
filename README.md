# Blinkit-Dashboard
## Recommended Structure and order
#### BlinkAnalytics: Quick Commerce Sales Insights Dashboard
A dynamic, interactive data visualization tool designed to analyze Blinkit’s retail sales performance, focusing on outlet distribution, product categories, customer behavior, and overall business insights.

The BlinkAnalytics Dashboard is a visually engaging and analytical Power BI report built to explore Blinkit’s fast-commerce retail data. It focuses on key metrics such as total sales, item distribution, outlet segmentation, and customer ratings, helping stakeholders understand business performance and identify growth opportunities.

#### Tech Stack
The dashboard was built using the following tools and technologies:

📊 Power BI Desktop – Main data visualization platform used for report creation.

🔄 Power Query – Data transformation and cleaning layer for reshaping and preparing the data.

🧮 DAX (Data Analysis Expressions) – Used for calculated measures, KPIs, and dynamic visuals.

🗂️ Data Modeling – Relationships established among tables to enable cross-filtering and aggregation.

📁 File Format – .pbix for development and .png for dashboard previews.

#### Data Source
The dataset used for this dashboard is sourced from Kaggle.com and contains Blinkit’s retail sales data with over 8,500+ records of product-level information. It includes data across 4 outlet types (Grocery Store, Supermarket Type 1, Type 2, and Type 3), categorized by 3 location tiers (Tier 1, Tier 2, Tier 3) and 3 outlet sizes (Small, Medium, High). The dataset covers 15+ product categories such as Fruits, Snacks, Dairy, and Household items, with total recorded sales of approximately $1.2 million and an average customer rating of around 3.9. Key attributes include item type, fat content, sales, item visibility, outlet size, and location, enabling multi-dimensional analysis of sales performance, customer preferences, and outlet efficiency.

#### Features
• Business Problem

Quick commerce businesses like Blinkit generate large volumes of sales and operational data. However, extracting meaningful insights from this data to improve performance, optimize inventory, and understand customer behavior remains a challenge.

• Goal of the Dashboard

To create an interactive and user-friendly dashboard that provides insights into sales performance, outlet efficiency, and product demand, enabling stakeholders to make data-driven decisions.

• Walkthrough of Key Visuals

KPI Cards (Top Panel)
Displays Total Sales ($1.2M), Number of Items (9K), Average Sales ($141), and Average Rating (3.92)

Outlet Establishment Trend (Line Chart)
Shows sales trends over years, identifying growth patterns and fluctuations in outlet performance

Fat Content Analysis (Donut Chart)
Compares Low Fat vs Regular product sales contribution

Item Type Analysis (Bar Chart)
Highlights top-performing product categories like Fruits, Snacks, and Household items

Outlet Size Distribution (Donut Chart)
Breaks down sales contribution across Small, Medium, and High outlet sizes

Outlet Location Analysis (Bar Chart)
Compares performance across Tier 1, Tier 2, and Tier 3 cities

Outlet Type Table
Provides detailed comparison of sales, number of items, ratings, and visibility across different outlet types

• Business Impact & Insights

📈 Tier 3 outlets generate the highest revenue contribution

🏪 Supermarket Type 1 dominates in both sales and item count

🥗 Regular fat products outperform low-fat products in sales

🍎 Fruits & Snack Foods are the top-selling categories

⭐ Average ratings are consistent across outlet types, indicating stable customer satisfaction

📊 Sales trends show initial growth followed by stabilization
